# Scripts-For-PythonScripter-Extension-

A collection of practical Python Scripter scripts for Burp Suite to automate common web penetration testing tasks.

## Requirements
- Burp Suite
- Python Scripter Extension

## Available Scripts

### Script 1: [JWT Analyzer]
Automatically decodes JWT tokens, extracts header and payload information, and highlights token expiration details.

### Script 2: [Sensitive Data Finder]
Scans responses for potentially sensitive information such as API keys, JWTs, email addresses, and other interesting patterns.

### Script 3: [Response Diff Analyzer]
Compares responses to identify differences in status codes, response lengths, and content changes.

### Script 4: [Reflected Input Detector]
Detects when user-supplied input is reflected back in the response, helping prioritize further manual testing.

### Script 5: [Security Headers Auditor]
Checks for common security headers and provides a quick PASS/FAIL overview.


## Installation
1. Install Python Scripter Extension.
2. Copy the script code.
3. Paste it into Python Scripter.
4. Run the script.
